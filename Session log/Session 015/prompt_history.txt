## The pattern

Import at top of file:
```ts
import { reportSupabaseError } from '../lib/errorReporter'
```

For Supabase `.from(...).update/insert/delete` errors:
```ts
if (error) {
  reportSupabaseError(error, { component: 'ComponentName', action: 'actionName' })
  toast.error(error.message)
  return
}
```

For catch blocks:
```ts
} catch (err) {
  const e = err as Error
  // if already calling toast.error, just add reportError before it
  reportSupabaseError({ message: e.message }, { component: 'ComponentName', action: 'actionName' })
  toast.error(e.message || 'Failed to ...')
}
```

Only add `reportSupabaseError` on MUTATION errors (insert/update/delete that already call `toast.error`). Do NOT add it to load errors (fetch functions that set page-level error state).
