DELIBERATE DESIGN DECISIONS
----------------------------

  - Journey editor workspace uses ResizeObserver + absolute positioning
    rather than a fixed pixel canvas. This means the workspace adapts
    to different screen sizes and sidebar open/close states.

  - Grid snap default is 24px (Medium) instead of the old 48px. This
    gives better positioning control while still being visible as dots.

  - Query limit of 1000 is a safety valve, not pagination. If an org
    exceeds 1000 records in a list, they'll see the first 1000. Full
    pagination UI would be a future feature.

  - Console statements were removed entirely rather than gated behind
    a DEBUG flag. The error reporter (reportError) handles persistent
    error logging to the error_reports table, which is more useful
    than browser console for production debugging.
