CHANGES MADE

1. LOADING BUG — FINALLY FIXED (commit 16949d9)
   - Root cause: Supabase JS SDK auth-lock deadlock across modules.
     Any module using the SDK client could hang waiting on the auth
     lock under certain cold-start/race conditions, producing the
     "Request timed out. Please refresh the page." error on
     engagements, courses, mentees, etc.
   - Fix: page loads now go through raw REST calls via
     supabaseRestCall, bypassing the SDK auth lock for reads. SDK is
     still used where auth state is required.
   - This affects all modules from admin and mentee logins.
